# Учебный сайт-портфолио Loftschool
## тема Горы

> gulp-сборка работает на gulp версии 4.0. 

#### Для начала работы

1. ```clone this repo```
2. ```cd path/to/...```
3. ```gulp```

> Для отдельных команд используйте ```gulp taskname``` (полный перечень тасков можно посмотреть в файле ```tasks.js```)
